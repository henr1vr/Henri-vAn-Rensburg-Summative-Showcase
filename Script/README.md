# Popup function
The popups give a brief overview of each project in the My Work section. It almost acts as a brief introduction.

# User action
Each individual project's popup will appear when the user hovers over the project card for that specific project.

# Reason for intraction
I chose the hover interaction because it ensures that this extra content is findable by the user. If the trigger was a click, the user may never even know that there is more content to explore behind each card.
